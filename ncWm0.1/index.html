这个软件是做什么的？

​ 功能（目前）：将网易云音乐中的歌单数据 以m3u8格式 导入索尼Walkman随身听.

概况

谁用得到这个软件？

​ Walkman + 网易云音乐用户

​ 如果你不属于这个范围 但是也能耐心读完这篇文章 那很感谢你哦(๑•̀ㅂ•́)و✧

为什么选择这样的搭配？

​ 网易云音乐在社交方面一直是较为领先的， 其歌单也理应是质量最优的.

​ 个人选择MP3听歌的原因是不希望被手机上的各种广告/推荐分心， 那么自然也要选择系统纯净的MP3. 这时候，连显示时间功能都没有的索尼WALKMAN自然成为了不二之选.

为什么不采用分文件夹方式存储歌单？

​ 这样做会导致同一首歌重复存储好几次，数据冗余；其次，无法指定歌曲播放顺序往往会破坏歌单的美感，或是与习惯中的歌曲连续性相冲突.

使用

下载链接

​ (Github直下， 可能会比较慢)

使用方法

  1. 使用软件时 需将WALKMAN连接电脑.
  2. 需要把所有歌单中用到的音乐放到一个文件夹中，再把这个文件夹放入WALKMAN的MUSIC目录下 (建议使用Cloudmusic文件夹)
  3. 将2中提到的文件夹打开，点击地址栏最左端的小图标，复制文件夹地址，黏贴到软件中.
  4. 得到所要歌单的id. (若是客户端，可以点击通过链接分享， 随便找个地方黏贴， id后面的那串数字就是. 注意不是userid)
  5. 将歌单id复制到软件中.
  6. 等待软件处理 将会自动显示处理状态.
  7. 生成的m3u8会放在软件同目录地址, 将其复制到WALKMAN中MUSIC下即可.

注意事项

  1. 请不要将软件放在没有读写权限的路径 （如C盘根目录）
  2. 歌曲的命名方式必须为 “歌手 - 歌名”. ( PC/UWP网易云音乐软件直接下载的曲目都可以兼容 )

Coding

代码环境

  • Visual Studio 2017
  • .Net C#
  • Newtonsoft.Json

代码思路

  • 通过HttpRequest从官方API获得歌单JSON
  • 通过JObject以及JArray解析JSON
  • 通过StreamWriter写入文件
